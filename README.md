# Battleship
Battleship game for Telegram API

/Ru-ru

Спецификация

Игра - морской бой. Правила читай ниже (или на википедии).
На два игрока, один из которых может быть ИИ или другой человек.
Предусматривает локализацию
Предусматривает порт через Telegram API.

Правила игры:
    Создаются два игрока
    У каждого игрока два поля: своё и поле для визуализации вражеского поля (исходя из получаемой информации 
через стрельбу).
    На своём поле каждый игрок размещает корабли:
        *Корабли не могут пересекаться
        *Корабли не могут стоять стык-в-стык, соприкасаться углами
        *Типы и кол-во кораблей:
            #### 1x; ### 2x; ## 3x; # 4x;
    После начинается сама игра: сначала стреляет игрок1(или определяется случайно)
    Ставится полученный результат на поле1 игрока2 и поле2 у игрока1.
    Если есть попадание и у противника остаются корабли - ход продолжается, причём вокруг уничтоженого (если уничтожен)
корабля ставятся отметки "мимо".
    Если промох, ставится отметка "мимо", если попадание - "попадание"
    Если промох, ход передаётся другому игроку.
    Если у игрока не остаётся кораблей, игра заканчивается, ему присуждается "поражение", а другому - "победа".
